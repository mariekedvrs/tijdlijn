Tim Berners-Lee, CERN – 1989

Consultant software engineer Tim Berners-Lee was werkzaam bij CERN, het Europese instituut voor kernfysica. Wetenschappers die samenwerkten in de internationale projecten van CERN, moesten informatie uitwisselen via verschillende systemen en software. Dit kostte veel tijd en ergernissen. Informatie was moeilijk te vinden, verwees niet door en was gescheiden. Berners-Lee wilde de informatie-uitwisseling vergemakkelijken en kwam dus met een beter systeem, dat bovenop het internet draaide en er elementen van gebruikte zoals TCP en DNS. In maart 1989 schreef Berners-Lee zijn voorstel voor "a large hypertext database with typed links". Zijn supervisor zag potentie en schreef de tekst ‘Vague but exciting’ op zijn voorstel.

World Wide Web – 1990

Op 12 november 1990 verstuurden Tim Berners-Lee en Robert Cailliau de CERN-memo: WorldWideWeb: Proposal for a HyperText Project. Tegen kerst 1990 had Berners-Lee alle tools die noodzakelijk waren voor een werkend web gerealiseerd: de HyperText Transfer Protocol (HTTP), de HyperText Markup Language (HTML), de eerste Web browser (genaamd WorldWideWeb; dit was meteen ook een webeditor), de eerste HTTP server software, de eerste web server, en de eerste Webpagina's die het project beschreven.

Grafische webbrowser - 1993

In 1993 bracht het NCSA Mosaic uit. Dit was de eerste grafische webbrowser. Tot dan kon er alleen met een tekst georiënteerde browser (Lynx) gesurft worden. Daarna volgden al snel vele andere. De bekendste is Netscape Navigator; een aardige tweede is Microsoft Internet Explorer. Met de komst van een grafische webbrowser werd het mogelijk afbeeldingen te laten zien. Ook konden er allerlei dingen met tekst uitgehaald worden, die daarvoor niet mogelijk waren. Het internet werd hierdoor van het ene moment op het andere interessant voor het grote publiek. Het balletje ging pas echt rollen toen de media er lucht van kregen. Het werd ineens interessant om op het internet te adverteren, om erin te investeren.

De geboorte van WWW Consortium (WC3) – 1994

Het World Wide Web Consortium is een organisatie die de webstandaarden voor het wereldwijde web ontwerpt, zoals HTML, XHTML, XML, CSS en de Web Content Accessibility Guidelines. Het wordt geleid door Tim Berners-Lee, de originele bedenker van het HTTP-protocol en HTML, waar het web oorspronkelijk en nog steeds grotendeels op gebaseerd is. De organisatie is in 1994 opgericht in samenwerking met CERN, ondersteund door DARPA en de Europese Commissie.

De browser oorlog - 1995

In de jaren 90 ging de browseroorlog veelal tussen Internet Explorer en Netscape. Aanvankelijk had Netscape een groot marktaandeel. Bij het verschijnen van Internet Explorer in 1995 kwam hier langzaam een einde aan. Het steeds toenemende marktaandeel van Internet Explorer werd mede veroorzaakt door de bundeling met Microsoft Windows en de promotie van Internet Explorer door Microsoft. Rond de eeuwwisseling had Internet Explorer de browseroorlog vrijwel gewonnen. In de periode 2002-2003 bereikte Internet Explorer zo een marktaandeel van ongeveer 90%. Door het uitblijven van een nieuwe versie van Internet Explorer ontstonden er een aantal nieuwe browsers: Safari (2003) en Firefox (2002). Toen aan het eind van 2008 Google Chrome gelanceerd werd, verloor Internet Explorer ook marktaandeel aan deze browser. Later ging de groei van Google Chrome ook ten koste van het marktaandeel van Firefox. De vijf genoemde browsers hebben sinds medio 2008 nagenoeg de hele markt in handen. Internet Explorer en Google Chrome hebben vrijwel een gelijk marktaandeel verworven, terwijl Firefox op de derde plaats staat.

Websites worden interactief - 1996

De eerste versie van JavaScript werd in 1995 ontwikkeld door Brendan Eich van Netscape Communications Corporation voor gebruik in Netscape Navigator. Aanvankelijk was de naam Mocha en vervolgens LiveScript. De taal werd hernoemd tot JavaScript in de tijd dat in de Netscape-browser ook ondersteuning voor Java-applets werd ingebouwd. Met de introductie van JavaScript ontstonden de eerste mogelijkheden om webpagina's interactief te maken. Dit werd later Dynamic HTML genoemd. Netscape heeft in een poging om JavaScript als een officiële norm erkend te krijgen en er toch zelf voldoende zeggingskracht over te behouden de taal laten normeren door de European Computer Manufacturers Association (ECMA), waarbij de naam om redenen betreffende het merkrecht is gewijzigd in ECMAScript. Microsoft heeft een eigen implementatie van JavaScript ontwikkeld onder de naam JScript en Adobe heeft een eigen versie genaamd ActionScript die gebruikt wordt in Flash.

Web 2.0 – 2003

Web 2.0 verwijst naar de ontwikkeling van het internet tot een communicatiemiddel, tot een medium met een hoge graad van mens-computerinteractie. Iedere gebruiker kan informatie zowel uploaden als downloaden, iemand hoeft daar geen webmaster voor te zijn. Tim O'Reilly bedacht de naam na het uiteen spatten van de internetzeepbel van het Web 1.0. Bij die aanvankelijke populariteit van het internet hoorde een 'nieuwe' economische euforie die eind jaren negentig in de internetzeepbel, of dotcomhype, eindigde.

Responsive design – 2011

Responsive webdesign is een benadering van webdesign waarbij de web developer streeft naar een optimale webervaring voor een breed scala aan apparaten (van desktop computerschermen tot mobiele telefoons). Het ontwerp van een responsive website schaalt mee met de afmetingen van een scherm, zonder in te leveren op leesbaarheid van tekst of bruikbaarheid van de gebruikersinterface. De term responsive webdesign is bedacht in 2010 door Ethan Marcotte op de weblog A List Apart. Marcotte merkte destijds op dat steeds meer bedrijven investeerden in een mobiele website om hun klanten ook mobiel een optimale ervaring te bieden. Maar door het groeiende aantal apparaten, met allen hun eigen resolutie en afmetingen, bleek één mobiele website geen optimale oplossing. In plaats van te ontwerpen in vaste afmetingen, een benadering die stamt uit het printtijdperk, merkte Marcotte op dat het veel praktischer was om te ontwerpen in percentages en andere relatieve eenheden.
